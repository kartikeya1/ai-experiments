Keyboard Shortcuts

  • Ctrl/Cmd + S – Save (download JSON)
  • Ctrl/Cmd + O – Load save file
  • Ctrl/Cmd + N – New trip
  • Ctrl/Cmd + D – Duplicate active trip
  • Ctrl/Cmd + Z – Undo
  • Ctrl/Cmd + Shift + Z – Redo
  • Ctrl/Cmd + M – Toggle Structured / Hybrid mode
  • Ctrl/Cmd + L – Toggle dark / light
  • Ctrl/Cmd + Alt + S – Add section
  • Ctrl/Cmd + Alt + N – Add item to focused section
  • Ctrl/Cmd + Shift + V – Switch layout view
